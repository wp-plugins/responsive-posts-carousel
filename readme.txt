== Installation ==
1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Responsive Posts Carousel' and install it
3. Go to Dashboard > Responsive Posts Carousel and set your desired settings
4. Use this shortcode [posts-carousel id="1"] in any page
5. Now visit that page
